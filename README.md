# Thinject

A super simple, easy to use, portable, decent-performing IoC container - just for the fun of it.

## Implemented features
 - [x] Lifetimes
	 - [x] Singleton
	 - [x] Transient
 - [ ] Registrations
	 - [x] Type
	 - [x] Instance
	 - [ ] Lambda
 - [ ] Constructor Parameters
	 - [x] `IEnumerable<T>`
	 - [ ] `Func<T>`
 - [ ] Child Containers
